Small Cosy Rooms Hybrid RTX

EN - Small Cosy Rooms Hybrid RTX is a 3d realtime rendering made with Unreal Engine. This project was intended to use the advantages of both rendering methods in Unreal Engine 4: Raytracing for reflections/translucency and GPU Lightmass (from Luoshuang) for GI.

Based on real photographies references, the style of that home is modern and contemporary. The scene will be available with 2 levels: a Raytracing map for supported GPU and another "Rasterize" map with pre-calculated reflections for any GPU.

FR - Small Cosy Rooms Hybrid RTX est un rendu vidéo en temps-réel avec Unreal Engine. Ce projet avait pour but d'utiliser les avantages des deux méthodes de rendu dans Unreal Engine 4: Raytracing pour les réflections/translucence et GPU Lightmass (créé par Luoshuang) pour la GI.

La scène sera disponible avec 2 niveaux: un niveau avec Raytracing activé pour les cartes graphiques supportés, et un autre niveau "Rasterize" avec des reflections pré-calculées pour tout types de cartes graphiques.
