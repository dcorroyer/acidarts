Schooldesk Photography Reproduction

EN - Schooldesk Photography Reproduction is a still image made in Unreal Engine 4.

This is a photography reproduction. The goal of this quick project was to demonstrate that we can achieve realistic results using a realtime engine like Unreal Engine 4.

Softwares used: 3ds Max, Photoshop, Substance Painter and Unreal Engine 4.
